## Key Features

### 1. Vertically Aligned Stats
- All stats boxes are in the same horizontal row
- Makes comparison across methods easy
- Table-like appearance

### 2. Fixed Height Responses
- Responses have max-height of 400px
- Scrollable if content is longer
- Ensures stats alignment regardless of response length

### 3. Clean Design
- No background colors in stats area
- Simple borders and spacing
- Minimal but clear visual hierarchy

### 4. Parallel Execution
- All methods start simultaneously
- Results populate as they complete
- Real-time updates in each column

## Layout Flow

1. **Header Row**: Method names + descriptions
2. **Response Row**: Scrollable boxes with generated text
3. **Divider**: Horizontal line separating content from stats
4. **Stats Row**: Vertically aligned metrics (table-like)
